# QA Automation Project

This project demonstrates automated testing in Java using Selenium.  
It contains sample test scripts and examples to practice QA automation concepts.

## Technologies
- Java
- Selenium WebDriver
- TestNG / JUnit

## Project Structure
- `/src`       -> Source code
- `/tests`     -> Automated test scripts
- `/drivers`   -> Browser drivers

## Running Tests
To run the automated tests:

```bash
mvn test
```
Or run the test classes from your IDE.

## Objective
- Learn and practice QA automation techniques
- Understand how to write automated test scripts
- Apply best practices for testing web applications
